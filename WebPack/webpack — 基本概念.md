## webapck 基本概念
##### 基于 node.js 的开发、测试、编译、打包工具。
##### 通过 webpack 可以将多个资源模块打包成一个 js 文件。


---
#### webpack-dev-server
##### webpack 用于开发的本地服务器插件

```
// 启动 webpack-dev-server 并自动打开流浪器 并且指定 host 为 0.0.0.0（其他机器可以通过本机 IP 来访问，不然只能是 localhost 访问）
 webpack-dev-server --open --host 0.0.0.0     
```
